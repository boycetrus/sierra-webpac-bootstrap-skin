Request a book

Use this form to request books and other items via Interlibrary Loan. To request articles from journals and magazines please use the Journal article request form.

Important Notes

  • Read the information about the Interlibrary Loan Service on our website before making a request.
  • A non-refundable $5.70 is charged for all interlibrary loan requests.
  • The Library always attempts to obtain items from libraries that don't charge us first.
  • If not available from a non-charging location, $32.40 is the standard fee other libraries charge. Please indicate whether you are willing to pay $32.40 to borrow from a charging location in the form below.
  • A response from the supplying library is expected within 4 working days.

Do you have a deadline?

If you have a deadline, please discuss your request with library staff before making a request online. Visit or telephone the Library on 02 9710 0351 between 9am-5pm weekdays.
